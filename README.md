# Fault-detection-project

Course Project
Fault Detection and Diagnosis in Engineering Systems, 2026
Important Information:
- It is recommended that the project be carried out in groups.
- The students are responsible for forming groups to carry out the project.
- The groups should consist of a maximum of 5 students.
- Feel free to make contact to arrange supervisory meetings if required.
- Each group needs to submit a project report along with possible MATLAB programming files as an outcome of the project
- The deadline for submission is May 25th, 2026 at 23:59.


Fault Detection and Diagnosis of a Multiphase Flow Facility

The project is focused on FDD in Cranfield Multiphase Flow Facility (Figure 1). The details about this facility
are explained in the following paper:
- C. Ruiz-Cárcel, Y. Cao, D. Mba, L.Lao and R. T. Samuel, Statistical Process Monitoring of a Multiphase Flow Facility, Control Engineering Practice, V. 42, PP. 74–88, 2015,
http://www.sciencedirect.com/science/article/pii/S0967066115000866#

The data collected from the facility can be obtained from (for your convenience it is already uploaded in the project folder):
- http://www.mathworks.com/matlabcentral/fileexchange/50938-a-benchmark-case-for-statisticalprocess-
monitoring-cranfield-multiphase-flow-facility?s_cid=ME_prod_FX

The tasks in the project is to first understand the case study and then to implement at least two methods
for fault detection on Cranfield Multiphase Flow Facility. You are not restricted to the methods which are
discussed in the lectures; however, you should be able to explain the methods which you use.
If you would like to choose another case study instead of Cranfield Multiphase Facility, it is fine, however,
you should present your case study sufficiently in your report and submit the data along with the report.
